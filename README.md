# 📊 [sugarcane-production-analysis]

Analisis & Prediksi Produksi Tebu dalam 10 tahun kedepan menggunakan metode SARIMA.

## 🎯 Tujuan
Eksplorasi data historis produksi tebu Indonesia dan prediksi produksi beberapa tahun ke depan.

## 📁 Struktur Project
\```
├── data/
│   ├── raw/          # Data asli dari Kaggle
│   └── processed/    # Data setelah cleaning
├── notebooks/        # Jupyter notebooks analisis
├── src/              # Script Python
├── output/figures/   # Hasil visualisasi
└── requirements.txt
\```

## 🛠️ Tools & Library
- Python 3.x
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- Scikit-learn
- MySQL / SQLAlchemy

## 🚀 Cara Menjalankan
\```bash
# Clone repo
git clone https://github.com/username/nama-repo.git
cd nama-repo

# Buat virtual environment
python -m venv .venv
source .venv/bin/activate  # Linux/Mac

# Install dependencies
pip install -r requirements.txt

# Jalankan Jupyter
jupyter notebook
\```
